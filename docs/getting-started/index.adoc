== Getting Started

New to Pilaf? Start here to get up and running quickly.

=== Prerequisites

Before you begin, ensure you have:

* **Node.js 18+** - JavaScript runtime
* **pnpm 10+** - Package manager
* **PaperMC Server** - Running server with RCON enabled
* **Docker** (optional) - For containerized testing
